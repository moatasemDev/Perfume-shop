# متجر إيلوفان (Ilovan) للعطور الفاخرة

هذا المشروع هو متجر إلكتروني كامل تم تطويره باستخدام تقنيات الويب الأساسية:

*   **الواجهة الأمامية (Frontend):** HTML5, CSS3, JavaScript
*   **الواجهة الخلفية (Backend):** PHP (مع استخدام MySQLi)
*   **قاعدة البيانات:** MySQL



---

## 1. هيكل المشروع

يتكون هيكل المجلدات الرئيسي للمشروع من التالي:

```
ilo_store/
├── config/
│   └── db.php              # ملف إعدادات الاتصال بقاعدة البيانات
├── includes/
│   ├── functions.php       # جميع دوال PHP الأساسية (المنتجات، المقالات، السلة، المستخدمين)
│   ├── header.php          # ملف رأس الصفحة (Header)
│   └── footer.php          # ملف تذييل الصفحة (Footer)
├── public/                 # المجلد العام (Document Root)
│   ├── css/
│   │   └── style.css       # ملف تنسيق CSS الرئيسي
│   ├── js/
│   │   └── script.js       # ملف JavaScript للوظائف التفاعلية
│   ├── images/             # مجلد الصور (صور المنتجات والمقالات)
│   ├── index.php           # الصفحة الرئيسية
│   ├── products.php        # صفحة عرض جميع المنتجات
│   ├── product.php         # صفحة تفاصيل المنتج
│   ├── blog.php            # صفحة المدونة
│   ├── article.php         # صفحة تفاصيل المقال
│   ├── cart.php            # صفحة سلة التسوق
│   ├── checkout.php        # صفحة إتمام الدفع
│   ├── login.php           # صفحة تسجيل الدخول
│   ├── register.php        # صفحة تسجيل حساب جديد
│   ├── logout.php          # معالج تسجيل الخروج
│   ├── cart_action.php     # معالج إجراءات السلة (إضافة، حذف، تحديث)
│   ├── add_review.php      # معالج إضافة التقييمات
│   └── search.php          # صفحة نتائج البحث
└── database_schema.sql     # مخطط قاعدة البيانات SQL
└── README.md               # ملف التوثيق الحالي
```

##  الميزات الرئيسية

| الميزة | الملفات المعنية | الوصف |
| :--- | :--- | :--- |
| **عرض المنتجات** | `products.php`, `product.php` | عرض قائمة المنتجات وتفاصيل كل منتج مع إمكانية إضافة تقييمات. |
| **إدارة السلة** | `cart.php`, `cart_action.php` | استخدام جلسات PHP (`$_SESSION`) لتخزين عناصر السلة، مع إمكانية التعديل والحذف. |
| **إتمام الدفع** | `checkout.php` | نموذج بسيط لإدخال عنوان الشحن واختيار طريقة الدفع، وإنشاء طلب في قاعدة البيانات. |
| **المدونة** | `blog.php`, `article.php` | عرض قائمة المقالات وتفاصيل كل مقال. |
| **المصادقة** | `login.php`, `register.php`, `logout.php` | نظام أساسي لتسجيل الدخول والتسجيل والخروج للمستخدمين. |
| **البحث** | `search.php` | وظيفة بحث بسيطة في أسماء المنتجات وعناوين المقالات. |

##  ملاحظات التطوير

*   **الأمان:** تم استخدام `mysqli_prepare` و `mysqli_stmt_bind_param` لمنع هجمات حقن SQL.
*   **كلمات المرور:** يتم تخزين كلمات المرور مشفرة باستخدام `password_hash()` و `password_verify()`.
*   **التصميم:** تم تطبيق تصميم متجاوب (Responsive) باستخدام CSS لضمان عرض جيد على مختلف الأجهزة.
